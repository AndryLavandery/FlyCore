# FlyCore

Репозиторий теперь содержит каркас мода **FlyCore CCTV** для Minecraft **1.20.4 Forge** (шаг 1 реализации):

- настроена Gradle-конфигурация Forge;
- зарегистрированы базовые блоки, предметы, block entity, menu/screen;
- добавлены стартовые ресурсы (models/blockstates/lang/mod metadata).
